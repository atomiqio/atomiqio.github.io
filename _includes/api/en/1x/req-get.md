<h3 id='req.get'>req.get(field)</h3>

Returns the specified HTTP request header field (case-insensitive match).
The `Referrer` and `Referer` fields are interchangeable.

{% highlight js %}
req.get('Content-Type');
// => "text/plain"

req.get('content-type');
// => "text/plain"

req.get('Something');
// => undefined
{% endhighlight %}

Aliased as `req.header(field)`.
